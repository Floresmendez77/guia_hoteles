# 🏨 Guía Hoteles Honduras

Proyecto de estudio: Bootstrap 5 · jQuery · Sass · LESS

## 📁 Estructura

guia_hoteles/
├── index.html
├── js/
│   └── app.js
├── sass/
│   └── main.scss
├── less/
│   └── estilos.less
├── css/
│   └── styles.css
└── README.md

## 🔧 Compilar

**Sass**
sass --watch sass/main.scss:css/styles.css

**LESS**
lessc less/estilos.less css/styles.css

## 🛠️ Tecnologías
- Bootstrap 5 — Modal, Carousel, Tooltips
- jQuery — Render dinámico desde JSON
- Sass — Variables, mixins, nesting
- LESS — Variables @, mixins, operaciones
